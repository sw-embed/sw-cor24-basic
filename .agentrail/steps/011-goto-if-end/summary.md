Added GOTO, IF...THEN, END, and STOP. Dispatch is now a small re-entry loop driven by an `rd` flag: IF evaluates the condition with p_expr(4), consumes THEN, and sets rd=1 on true so the remainder of the line dispatches as a fresh statement. GOTO reuses store_find() to resolve the target line number and assigns tl directly; the main loop picks up the new position on the next iteration. END and STOP both clear the run flag (mi:=0), returning to the REPL. Error 3 = undefined line.

Trimmed 8 currently-unused keyword table entries (INPUT, GOSUB, RETURN, FOR, TO, STEP, NEXT, LIST) from `ik` to stay under the ~8KB UART monolith ceiling. These will be restored once the unit-system split (steps 018/019) removes the size constraint; the tokens they reserve are still valid, they just won't tokenize as keywords until then.
